    Caixa de Ferramentas JSON

    Ferramentas JSON

    Limpar e filtrar JSON para uso posterior.

    {{ jsonTools  | filter: valor | filterType:  }}

    Exemplo

    • Filtro: Um valor pelo qual devemos filtrar itens.
    • Tipo de Filtro: Se definido como Igual, filtre por objetos/chaves onde o valor é igual ao valor definido em filtro. Se definido como Não Igual, filtre por objetos/chaves onde o valor não é igual ao valor definido em filtro.

    Sucesso

    • O JSON resultante.
    Nota 
    {
  livros: [
{
      título: O Grande Gatsby,
      resumo: O Grande Gatsby é um romance de F. Scott Fitzgerald que acompanha a história de Jay Gatsby, um homem rico e misterioso, e sua busca pelo Sonho Americano. Ambientado nos anos 1920, o livro explora temas de amor, riqueza e a corrupção do Sonho Americano.,
      autor: F. Scott Fitzgerald
}
]
}
{{ jsonTools  | filter: O Grande Gatsby | filterType: Igual }}

    Resultaria em:

    {
  livros: [
{
      título: O Grande Gatsby
}
]
}

    Onde definir filterType como Não Igual resultaria em:

    {
  livros: [
{
      resumo: O Grande Gatsby é um romance de F. Scott Fitzgerald que acompanha a história de Jay Gatsby, um homem rico e misterioso, e sua busca pelo Sonho Americano. Ambientado nos anos 1920, o livro explora temas de amor, riqueza e a corrupção do Sonho Americano.,
      autor: F. Scott Fitzgerald
}
]
}

    Remapear JSON

    Remapear elementos em um objeto JSON de um nome de chave para outro.

    {{ reMapJSON  | match:  | replace:  }}

    Exemplo

    • Correspondência: Uma variável que você está tentando substituir.
    • Substituir: A variável que substituirá todas as instâncias da variável definida como seu parâmetro Correspondência.

    Sucesso

    • Altera o nome da chave de uma variável para o nome usado no parâmetro Substituir.
    Nota 
    {
  livros: [
{
      título: O Grande Gatsby,
      resumo: O Grande Gatsby é um romance de F. Scott Fitzgerald que acompanha a história de Jay Gatsby, um homem rico e misterioso, e sua busca pelo Sonho Americano. Ambientado nos anos 1920, o livro explora temas de amor, riqueza e a corrupção do Sonho Americano.,
      autor: F. Scott Fitzgerald
}
]
}
{{ reMapJSON  | match: O Grande Gatsby | replace: To Kill a Mockingbird }}

    Resultaria em:

    {
  livros: [
{
      título: To Kill a Mockingbird,
      resumo: O Grande Gatsby é um romance de F. Scott Fitzgerald que acompanha a história de Jay Gatsby, um homem rico e misterioso, e sua busca pelo Sonho Americano. Ambientado nos anos 1920, o livro explora temas de amor, riqueza e a corrupção do Sonho Americano.,
      autor: F. Scott Fitzgerald
}
]
}

    Filtro de Array JSON

    Filtrar um array JSON

    {{ arrayFilter  | filter:  | filterType:  }}

    Exemplo

    • Filtro: Um valor pelo qual devemos filtrar itens.
    • Tipo de Filtro: Existem quatro tipos de índice diferentes:

    1. Índice: Você pode usar um filterType de Índice e passar o índice numérico da matriz para retornar.

    2. Intervalo de Índices: IndexRange espera número-número de índices a extrair, por exemplo: 1-3.

    3. Correspondência de Valor: Correspondência de valor e Contém valor filtrarão a matriz encontrando objetos na matriz com propriedades que correspondem ao valor do filtro.

    4.

    Valor Contém: O valor correspondente e o valor Contém filtrarão a matriz encontrando objetos na matriz com propriedades que correspondam ao valor do filtro.

    Filtro de Chave JSON

    Filtre um Objeto JSON por uma lista de chaves.

    {{ keyFilter | filter: }}

    Example

    • Filtro: Uma lista separada por vírgulas de chaves para filtrar um objeto JSON.

    Success

    • O objeto JSON, filtrado pelas chaves inseridas nos parâmetros.
    Nota 
    {
      title: The Great Gatsby,
      summary: The Great Gatsby is a novel by F. Scott Fitzgerald that follows the story of Jay Gatsby, a wealthy and mysterious man, and his pursuit of the American Dream. Set in the 1920s, the book explores themes of love, wealth, and the corruption of the American Dream.,
      author: F. Scott Fitzgerald
}
{{ keyFilter | filter: title, author }}

    Resultaria em:

    {title:The Great Gatsby,author:F. Scott Fitzgerald}

    JSON para Variáveis

    Aceita JSON como entrada, achatando as chaves do objeto e definindo essas chaves como variáveis no contexto do mAIstro.

    Example

    Nenhum - Os dados devem ser encadeados nesta função.

    Success

    Nenhum - As variáveis são atribuídas como resultado desta função.

    Nota

    Os dados podem vir de um LLM, um arquivo, uma resposta da API REST, etc.

    {{ LLM | prompt: Output some information about a book in JSON format. Include title, summary, and author. | modelCard: }}=>{{ jsonToVars }}

    A saída do LLM:

    {
  title: The Great Gatsby,
  summary: The Great Gatsby is a novel by F. Scott Fitzgerald that follows the story of Jay Gatsby, a wealthy and mysterious man, and his pursuit of the American Dream. Set in the 1920s, the book explores themes of love, wealth, and the corruption of the American Dream.,
  author: F. Scott Fitzgerald
}

    Variáveis para JSON

    Converte variáveis de ambiente em JSON.

    {{ varsToJSON | path: | variable: }}

    Example

    • Caminho: (Opcional) O caminho achatado a partir do qual começar a obter valores.
    • Variável: O nome da variável para atribuir o JSON resultante.

    Success

    Nada - A saída é atribuída ao nome da variável definida.

    Nota 
    Howard has 20 cats and 40 dogs. 
He took them to the vet last week.=>{{ grammar }}=>{{ variable | name: text }}
{{ varsToJSON | path: | variable: gm }}
<< name: gm, prompt: false >>

    Resultará em:

    {
grammar: {
year: [
2024
],
context: ,
dates: [
last,
week
],
propernouns: [
Howard
],
nouns: [
20 cats,
40 dogs,
vet,
week
],
preps: [
He,
them
],
determiners: []
},
text: Howard has 20 cats and 40 dogs. \nHe took them to the vet last week.
}
    Nota

    Usando o parâmetro path, podemos especificar o caminho inicial dos valores que queremos:

    Howard has 20 cats and 40 dogs. 
He took them to the vet last week.=>{{ grammar }}=>{{ variable | name: text }}
{{ varsToJSON | path: grammar.dates | variable: gm }}
<< name: gm, prompt: false >>

    Resultará em:

    {
grammar: {
dates: [
last,
week
]
}
}
