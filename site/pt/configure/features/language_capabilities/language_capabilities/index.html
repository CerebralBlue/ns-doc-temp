    Capacidades Linguísticas

    Identificar Idioma

    O que é?

    • NeuralSeek fornece um serviço que analisaria e identificaria o idioma de um determinado texto.

    Por que é importante?

    • Qualquer aplicativo que precise entender em que idioma um determinado texto está pode agora usar o NeuralSeek para fazê-lo, em vez de depender de outros serviços externos.

    Como funciona?

    • A identificação de idioma é fornecida como API REST e pode ser testada na documentação da API NeuralSeek. A carga útil da mensagem está no formato text/plain e contém text em determinados idiomas. Uma mensagem de exemplo ficaria assim:
    이 언어는 어떤 언어입니까?
    • O NeuralSeek então identificaria em que idioma isso está e retornaria o código do idioma e o escore de confiança:
    [
    {
        language: ko,
        confidence: 0.95
    }
]

    Categorização de Intenção

    O que é?

    • O NeuralSeek pode categorizar automaticamente a entrada e as perguntas do usuário em categorias. Essas categorias podem ser qualquer coisa - produtos, organizações, departamentos, etc. Os usuários podem configurar categorias na guia Configurar, inserindo nomes e descrições de categorias. Esses serão então usados ​​para corresponder a entrada do usuário às categorias. Entradas de usuário que não correspondem a nenhuma categoria, ou que correspondem muito de perto a várias categorias, serão colocadas em uma categoria padrão chamada Outro. Essa categoria padrão não pode ser modificada.

    Por que é importante?

    • A categorização é muito útil para dimensionar o NeuralSeek dentro de uma organização. Ao agrupar intents em categorias, pode facilitar muito para os especialistas no assunto agirem rapidamente em sua área específica de conteúdo. A categorização pode ser útil mesmo fora do contexto de responder a perguntas de usuários - por exemplo, no encaminhamento de perguntas de clientes para o departamento ou agente ao vivo correto. A categorização pode ser chamada diretamente via API.

    Como funciona?

    • A entrada do usuário é pontuada e agrupada com base no título e na descrição da categoria, e com base nos intents que foram movidos manualmente para as categorias (aprendizado próprio). Assim que a categorização for ativada, as telas Curate e Analytics mudarão para mostrar agrupamentos em torno de categorias. A categorização não é retroativa - isso significa que, se você definir uma nova categoria, não reprocessaremos automaticamente todas as entradas antigas do usuário nas novas categorias. Os usuários podem mover manualmente os intents para as categorias por meio da guia Curate ou dos recursos de download/edição de CSV. As edições feitas serão usadas para treinar o sistema para eventos de categorização futuros.

    Tradução de Idioma

    O que é?

    • O NeuralSeek fornece tradução de idiomas que permitirá que os usuários o chamem para traduzir idiomas em diferentes idiomas.

    Por que é importante?

    • Qualquer aplicativo que precise traduzir um determinado texto para outro idioma pode agora usar o NeuralSeek para fazê-lo, em vez de depender de outros serviços de tradução externos.

    Como funciona?

    • A tradução é fornecida como API REST e pode ser testada na documentação da API NeuralSeek.
    • A carga útil da mensagem está no formato JSON e contém uma matriz de text em determinado(s) idioma(s). Outro atributo é target que especifica o idioma de destino para o qual a tradução deve ser realizada. Uma mensagem de exemplo ficaria assim:
    {
    text: [
    NeuralSeek introduced several new features in July 2023, including streaming responses for web use cases, enhanced cross-lingual support, curate to CSV/upload curated QA from CSV, improved semantic match analysis, updated IBM WatsonX model compatibility, and AWS Lex round-trip monitoring.
    ],
    target: ko
},

    Para obter mais detalhes sobre quais códigos de idioma são compatíveis, consulte Suporte a vários idiomas. (es) na solicitação ao invocar Seek.

    O mesmo pode ser alcançado ao invocar Seek usando a API REST. Você pode especificar o idioma em options > language.

    Suporte multilíngue para KBs

    O NeuralSeek oferece suporte robusto a vários idiomas, permitindo que os usuários interajam com uma base de conhecimento (KB) em um idioma diferente daquele em que a KB está escrita. Isso é particularmente útil em cenários em que a base de conhecimento está em um idioma (por exemplo, inglês), mas os usuários precisam consultá-la em outro idioma (por exemplo, espanhol).

    Como Funciona

    Quando um usuário consulta a base de conhecimento em um idioma diferente, o NeuralSeek lida com o processo de tradução de forma transparente:

    1. Consulta do Usuário no Idioma Nativo: O usuário faz uma pergunta em seu idioma nativo (por exemplo, espanhol).
    2. Tradução para o Idioma da KB: O NeuralSeek traduz a pergunta do usuário para o idioma da base de conhecimento (por exemplo, inglês).
    3. Consultando a KB: A pergunta traduzida é usada para pesquisar a base de conhecimento.
    4. Recuperando a Resposta: O NeuralSeek recupera a resposta do LLM em seu idioma nativo.
    5. Entregando a Resposta: O usuário recebe a resposta em seu idioma nativo.

    Cenário de Exemplo

    Pergunta em espanhol, KB em inglês

    1. Consulta do Usuário: ¿Cuál es la capital de Francia?
    2. Traduzir para o Inglês: What is the capital of France?
    3. Consultar a KB em Inglês: O sistema pesquisa "What is the capital of France?" na base de conhecimento em inglês.
    4. Recuperar a Resposta do LLM em Espanhol: La capital de Francia es París.
    5. Entregar a Resposta: La capital de Francia es París.

    Para configurar o NeuralSeek para suporte multilíngue, siga estas etapas:

    Etapa 1: Configurar o Idioma da Base de Conhecimento

    • Navegue até a Guia de Configuração: Acesse as configurações do NeuralSeek.
    • Selecione o Idioma da Sua Base de Conhecimento: Escolha o idioma em que sua base de conhecimento está escrita (inglês, neste caso).
    • Salve a Configuração: Certifique-se de que suas configurações sejam salvas corretamente para aplicar as alterações.

    Etapa 2: Testar Consultas em Vários Idiomas

    • Vá para a Guia Seek: Acesse a interface de consulta do NeuralSeek.
    • Digite uma Pergunta em Espanhol: Teste a configuração inserindo uma pergunta em espanhol, como ¿Cuál es la capital de Francia?
    • Observe a Resposta: O NeuralSeek deve traduzir a pergunta, consultar a base de conhecimento em inglês e retornar a resposta no idioma desejado: La capital de Francia es París.
