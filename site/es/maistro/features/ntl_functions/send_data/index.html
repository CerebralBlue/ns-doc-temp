    Correo electrónico

    Conexión al servidor SMTP. Envíe fácilmente correos electrónicos. Particularmente útil en plantillas.

    {{ email|host: | port:  | user:  | pass:  | from:  | to:  | subject:  | message:  }}

    Ejemplo

    • Host: El nombre de host del servidor SMTP.
    • Puerto: El puerto del servidor SMTP.
    • Usuario y contraseña: Las credenciales para el servidor.
    • De: La dirección de correo electrónico del remitente.
    • Para: La dirección de correo electrónico del destinatario.
    • Asunto: El asunto del correo electrónico.
    • Mensaje: El contenido del cuerpo del correo electrónico.
