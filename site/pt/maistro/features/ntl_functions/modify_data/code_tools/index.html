    Caixa de Ferramentas de Código

    Extrair código

    Extraia apenas o código de uma string, removendo qualquer coisa extra, como comentários ou outro texto.

    Isso é útil quando você está limpando o código gerado por um LLM ou extraindo código de conteúdo misto, como scripts ou resultados de web scraping.

    {{ extractCode }}

    Example

    • Nenhum - Os dados da string devem ser fornecidos como entrada encadeada para esta função.

    Success

    • Descrição - Um breve resumo do código extraído.
    • Tipo - A linguagem de programação do código.
    • Código - O código limpo e extraído.
    Nota

    Aqui está como extrair código Python de uma saída de LLM:

    {{ LLM  | prompt: Crie um script Python para iterar sobre uma matriz de 3 frutas diferentes e imprimir seu nome e tamanho dos caracteres | cache: true }}
{{ extractCode  }}=>{{ variable  | name: extractedCode }}
<< name: extractedCode, prompt: false >>

    O resultado será o código extraído. Neste caso, seria Python:

    # Define uma matriz de frutas
frutas = [maçã, banana, cereja]

# Iterar sobre a matriz
for fruta in frutas:
    # Obter o comprimento do nome da fruta
    comprimento = len(fruta)
    # Imprimir o nome da fruta e seu tamanho de caractere
    print(fFruta: {fruta}, Caracteres: {comprimento})

    Limpar HTML

    Extraia apenas o HTML de uma string, removendo qualquer coisa extra, como comentários ou outro texto.

    Isso é útil quando você está limpando o HTML gerado por um LLM ou extraindo HTML de conteúdo misto, como resultados de web scraping.

    {{ cleanHTML }}

    Example

    • Seletores CSS - A matriz de seletores CSS a serem removidos do HTML.

    Success

    • Código HTML - O HTML limpo e extraído.
    Nota

    Aqui está como converter e extrair código HTML de uma saída de LLM:

    <html>
<head>
    <title>Amostra de HTML</title>
</head>
<body>
    <div class=header>
        <h1>Bem-vindo à Página de Amostra</h1>
    </div>
    <div class=content>
        <p>Este é um parágrafo de amostra com algum <span class=highlight>texto destacado</span>.</p>
        <p>Outro parágrafo com <a href=https://example.com>um link</a>.</p>
    </div>
    <div class=footer>
        <p>Conteúdo do rodapé aqui.</p>
    </div>
</body>
</html>
{{ cleanHTML  | selectors: ['.footer'] }}=>{{ variable  | name: cleanedHTML }}
<< name: cleanedHTML, prompt: false >>

    O resultado será o HTML extraído:

    <html>
<head>
    <title>Amostra de HTML</title>
</head>
<body>
    <div class=header>
        <h1>Bem-vindo à Página de Amostra</h1>
    </div>
    <div class=content>
        <p>Este é um parágrafo de amostra com algum <span class=highlight>texto destacado</span>.</p>
        <p>Outro parágrafo com <a href=https://example.com>um link</a>.</p>
    </div>
</body>
</html>

    Limpar SQL

    Extraia apenas o código SQL de uma string, criando uma versão formatada da consulta, se habilitado.

    Este recurso é útil para formatar e proteger o código SQL de respostas geradas por LLM, adaptado a diferentes tipos de servidores SQL.

    {{ cleanSQL  | reformat: true | onlySelect: false | dbType: PostgresQL }}

    Example

    • Reformatar - Quando definido como verdadeiro, o código SQL é reformatado para melhor legibilidade. Padrão é falso.
    • Apenas instruções SELECT - Quando definido como verdadeiro, apenas instruções SELECT são extraídas. O padrão é verdadeiro.
    • Tipo de banco de dados - Especifica o tipo de banco de dados (por exemplo, PostgresQL, MySQL, BigQuery) para compatibilidade durante a extração e limpeza de SQL.

    Sucesso

    • Código - O código SQL extraído e formatado.
    Nota

    Aqui está como limpar o SQL:

    {{ LLM  | prompt: Gere uma consulta SELECT para contar e agrupar os alunos por matrícula usando as tabelas de alunos, cursos e matrículas. Retorne apenas o código SQL, sem quaisquer explicações. | cache: true }}
{{ cleanSQL  | reformat: true | onlySelect: true | dbType: PostgresQL }}=>{{ variable  | name: cleanedSQL }}
<< name: cleanedSQL, prompt: false >>

    O resultado será o SQL extraído e formatado:

    SELECT c.course_name, COUNT(e.student_id) AS student_count FROM courses AS c INNER JOIN enrollments AS e ON c.course_id = e.course_id GROUP BY c.course_name
