    LLMs Suportados

    Visão Geral

    O NeuralSeek suporta LLMs de muitos provedores, incluindo:

    • Amazon Bedrock
    • Serviços Cognitivos da Azure
    • Google Vertex AI
    • HuggingFace
    • OpenAI
    • together.ai
    • watsonx.ai

    Além de quaisquer endpoints compatíveis com a OpenAI.

    💡 A escolha de LLM está disponível com o plano BYOLLM (traga seu próprio modelo de linguagem grande) da NeuralSeek.

    💡 Os LLMs podem variar em suas capacidades e desempenhos. Alguns LLM podem levar até 30 segundos e mais para gerar uma resposta completa. Use cautela ao usar em conjunto com uma plataforma de agente virtual que imponha um tempo limite estrito.

    Configurando um LLM

    ⚠️ Para configurar um LLM, certifique-se de ter se inscrito no plano Bring Your Own LLM (BYOLLM). Todos os outros planos usarão o LLM curado da NeuralSeek, e essa opção não estará disponível.

    1. Na interface da NeuralSeek, navegue até a página Configurar > Detalhes do LLM, usando o menu superior.
    2. Clique no botão Adicionar um LLM.
    3. Selecione a Plataforma e a Seleção de LLM. (por exemplo, Plataforma: Self-Hosted, LLM: Flan-u2)
    4. Clique em Adicionar.
    5. Digite a Chave da API do LLM no campo de entrada da Chave da API do LLM.
    6. Revise os Idiomas Habilitados (apresentados como seleção múltipla)
    7. Revise as funções do LLM disponíveis (apresentadas como caixa de seleção)
    8. Clique no botão Testar para verificar se a chave da API funciona.

    💡 Você deve adicionar pelo menos um LLM. Se você adicionar vários, a NeuralSeek fará o balanceamento de carga entre eles para as funções selecionadas que têm vários LLMs. Recursos que um LLM não é capaz de realizar serão desativados. Se você não fornecer um LLM para uma função, não haverá fallback e essa função da NeuralSeek será desativada.
