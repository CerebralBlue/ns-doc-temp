    Extraer código

    Extraer solo el código de una cadena, eliminando cualquier cosa extra como comentarios u otro texto.

    Esto es útil cuando se limpia el código generado por un LLM o se extrae código de contenido mixto como scripts o resultados de web scraping.

    {{ extractCode }}

    Example

    • Ninguno - Los datos de la cadena deben proporcionarse como entrada encadenada a esta función.

    Éxito

    • Descripción - Un breve resumen del código extraído.
    • Tipo - El lenguaje de programación del código.
    • Código - El código limpiado y extraído.
    Nota

    Así es como se extrae código Python de la salida de un LLM:

    {{ LLM  | prompt: Crea un script de Python para iterar sobre una matriz de 3 frutas diferentes e imprimir su nombre y el tamaño de los caracteres | cache: true }}
{{ extractCode  }}=>{{ variable  | name: extractedCode }}
<< name: extractedCode, prompt: false >>

    El resultado será el código extraído. En este caso sería Python:

    # Define una matriz de frutas
frutas = [manzana, banana, cereza]

# Iterar sobre la matriz
for fruta in frutas:
    # Obtener la longitud del nombre de la fruta
    longitud = len(fruta)
    # Imprimir el nombre de la fruta y su tamaño de carácter
    print(fFruta: {fruta}, Caracteres: {longitud})

    Limpiar HTML

    Extraer solo el HTML de una cadena, eliminando cualquier cosa extra como comentarios u otro texto.

    Esto es útil cuando se limpia el HTML generado por un LLM o se extrae HTML de contenido mixto como resultados de web scraping.

    {{ cleanHTML }}

    Example

    • Selectores CSS - La matriz de selectores CSS a eliminar del HTML.

    Éxito

    • Código HTML - El HTML limpiado y extraído.
    Nota

    Así es como se convierte en extraer código HTML de la salida de un LLM:

    <html>
<head>
    <title>Ejemplo de HTML</title>
</head>
<body>
    <div class=encabezado>
        <h1>Bienvenido a la página de ejemplo</h1>
    </div>
    <div class=contenido>
        <p>Este es un párrafo de ejemplo con algo de <span class=destacado>texto resaltado</span>.</p>
        <p>Otro párrafo con <a href=https://example.com>un enlace</a>.</p>
    </div>
    <div class=pie>
        <p>Contenido del pie de página aquí.</p>
    </div>
</body>
</html>
{{ cleanHTML  | selectors: ['.pie'] }}=>{{ variable  | name: htmlLimpio }}
<< name: htmlLimpio, prompt: false >>

    El resultado será el HTML extraído:

    <html>
<head>
    <title>Ejemplo de HTML</title>
</head>
<body>
    <div class=encabezado>
        <h1>Bienvenido a la página de ejemplo</h1>
    </div>
    <div class=contenido>
        <p>Este es un párrafo de ejemplo con algo de <span class=destacado>texto resaltado</span>.</p>
        <p>Otro párrafo con <a href=https://example.com>un enlace</a>.</p>
    </div>
</body>
</html>

    Limpiar SQL

    Extraer solo el código SQL de una cadena, creando una versión con formato de la consulta si está habilitado.

    Esta función es útil para dar formato y asegurar el código SQL de las respuestas generadas por LLM, adaptado a diferentes tipos de servidores SQL.

    {{ cleanSQL  | reformat: true | onlySelect: false | dbType: PostgresQL }}

    Example

    • Reformatear - Cuando se establece en true, el código SQL se reformatea para mejorar la legibilidad. El valor predeterminado es false.
    • Solo instrucciones SELECT - Cuando se establece en true, solo se extraen las instrucciones SELECT. El valor predeterminado es true.
    • Tipo de base de datos - Especifica el tipo de base de datos (por ejemplo, PostgresQL, MySQL, BigQuery) para la compatibilidad durante la extracción y limpieza de SQL.

    Éxito

    • Código - El código SQL extraído y formateado.
    Nota

    Así es como limpiar SQL:

    {{ LLM  | prompt: Generar una consulta SELECT para contar y agrupar a los estudiantes por inscripción en cursos utilizando las tablas de estudiantes, cursos e inscripciones. Devuelve solo el código SQL, sin ninguna explicación. | cache: true }}
{{ cleanSQL  | reformat: true | onlySelect: true | dbType: PostgresQL }}=>{{ variable  | name: cleanedSQL }}
<< name: cleanedSQL, prompt: false >>

    El resultado será el SQL extraído y formateado:

    SELECT c.course_name, COUNT(e.student_id) AS student_count FROM courses AS c INNER JOIN enrollments AS e ON c.course_id = e.course_id GROUP BY c.course_name
