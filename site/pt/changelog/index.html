    Registro de Alterações

    Outubro - 2024

    Novos recursos

    • Atualizações do mAIstro:
      • O NTL agora possui realce de código e rollup, além de um novo editor
      • Caixa de ferramentas de código: um conjunto de funções fáceis de um único nó para:
      • extrair código gerado / sql / html da maioria dos LLMs,
      • proteger, validar e reescrever SQL
      • Limpar HTML e extrair texto
    • Atualizações do Limpador HTML. O NeuralSeek limpa automaticamente os documentos HTML raspados nos KBs aos quais você se conecta. Agora você pode especificar seletores CSS para remover, além de nossa limpeza normal, bem como desativar o limpador.
    • Governança - Insights de custos. Ambos os lados da governança recebem uma nova guia que compara o custo de seus modelos selecionados em todos os outros modelos para os quais temos dados de capacidade e custo.
    • DQL para descoberta elástica / watsonx. Trouxemos nosso interpretador DQL para o elástico, para que você possa passar filtros DQL e fazer facilmente filtragem complicada e reduzir os riscos de migração ao vir da descoberta.

    Setembro - 2024

    Novos recursos

    • Construtor visual de vários agentes. Ative o vários agentes na guia de configuração e crie facilmente fluxos de vários agentes orientados por categoria. Sem codificação necessária! Cada nó da árvore de vários agentes pode ter sua própria configuração (kb, llm's, tudo) e guardrails. Também unificamos os lados Seek e mAIstro da casa, para que você possa chamar ambos a partir de qualquer api.
      • Cada nó pode ser de um tipo de busca tradicional ou de um novo nó liderado pelo mAIstro. Os nós do mAIstro enviam intenções que os atingem para uma ação padrão, em vez de criar uma nova intenção. Isso permite que você faça desambiguação ou direcione o usuário para os recursos que você habilitou - como abrir tíquetes de problemas ou outras ações lideradas pelo mAIstro
      • Qualquer intenção pode executar diretamente um fluxo do mAIstro. Então, para uma pergunta como "como está o tempo hoje", você poderia chamar um fluxo do mAIstro em vez de enviá-la para o caminho/kb de busca tradicional.
      • Agora você pode adicionar uma nova intenção diretamente via guia de configuração e curadoria.
      • Os guardrails podem executar fluxos do mAIstro. Confiança mínima, palavras mínimas e máximas podem todas executar fluxos personalizados do mAIstro para fornecer respostas específicas ao contexto e ao idioma quando esses guardrails forem atingidos.
    • Bate-papo! Introduzimos uma interface semelhante ao ChatGPT, bem como um SDK e código de incorporação. Você pode adicionar rapidamente um agente virtual ao seu site simplesmente inserindo o código de incorporação. O SDK de bate-papo permite arrastar e soltar imagens e arquivos - então você poderia facilmente construir um bot que permita que os usuários façam perguntas fornecendo imagens, como "Quero um refrigerador como este".
    • OCR! Lançamos nossos recursos de OCR há algumas semanas, mas nunca os anunciamos. Agora temos OCR incorporado ao sistema. Quando você usa ou carrega um documento no mAIstro, automaticamente faremos OCR em todos os PDFs que encontrarmos que sejam baseados em imagem e não em texto. Você também pode fazer OCR em arquivos de imagem. Além disso, lançamos um modelo de OCR duplo - basicamente mostrando como aproveitar e paralelizar nosso OCR, além das capacidades visuais de um modelo multimodal, para fazer coisas incríveis para OCR de documentos complexos, mantendo a formatação original. Essa capacidade supera de longe as ferramentas de OCR legadas, com quase nenhum ajuste.
    • Geração de documentos - construímos uma nova geração de documentos por trás dos bastidores, permitindo maior capacidade de gerar arquivos PDF e Word bem formatados em escala.
    • LLMs! LLama3.2 tanto no bedrock quanto no watsonx.ai. Este é o primeiro modelo multimodal disponível no watsonx.
    • novo modelo curado! Para os planos PPA, o modelo curado 1.1 está disponível em todas as regiões
    • Atualizações de registro. Os registros de pesquisa foram movidos para a guia Governança e incluem ainda mais detalhes. Ativar o registro corporativo para a reprodução de transações - que se tornou rapidamente um must-have para as PMEs
    • novos nós mAIstro! Lançamos um kit de ferramentas XML, bem como cerca de uma dúzia de outros novos nós e conectores
    • melhorias na tradução! Por meio da API, você pode substituir o tamanho máximo do chunk que usamos - o que pode acelerar drasticamente as traduções para traduções de tamanho médio quando usar LLMs / plataformas de inferência lentas.

    Agosto - 2024

    Novos recursos

    • mAIstro Confiança Mínima - Em uma pesquisa, ao atingir seu limite de confiança mínima, execute um fluxo mAIstro personalizado. Você pode usar isso simplesmente para criar uma nota de "não sei" ciente do contexto - mas também pode usar isso para iniciar uma notificação, uma chamada de serviço externo ou um chamado de tíquete... Qualquer coisa, na verdade.
    • Insights Semânticos - agora no gráfico de termos alucinados, você pode clicar em um termo para permitir diretamente os itens...
    • Carregador de Dados - Arraste e solte arquivos em nosso novo carregador, que aproveita o mAIstro para dividir/carregar documentos. Você pode usar qualquer função ou integração mAIstro, fazer chamadas REST, gerar embeddings, fazer loop e dividir documentos automaticamente... Fornecemos um exemplo de carregador para elastic/watsonx discovery.
    • Governança para mAIstro!
      • Rastrear e fornecer insights automaticamente para todos os modelos mAIstro, filtráveis por modelo.
      • Os insights de fluxo ajudam a rastrear o tempo gasto em nosso mecanismo paralelo, ajudando você a otimizar os fluxos e entender onde eles estão gastando a maior parte de seu tempo.
      • Os insights de token espelham a guia de insights de token de Seek, ajudando a mostrar o consumo de token, o custo e as opções de comparação de modelos para os LLMs usados para alimentar seus fluxos mAIstro
    • Atualizações de Governança de Seek
      • Filtrar por filtro... Ao usar filtros em Seek, agora você pode rastrear automaticamente a governança pelo filtro aplicado.

    Julho - 2024

    Novos recursos

    • Novos LLMs
      • Mistral-large no watsonx.ai
      • GPT-4o-mini no OpenAI.
    • Endpoints de API de streaming para Seek e mAIstro para o assistente watsonx. Estes têm o tipo de conteúdo necessário na especificação OpenAPI. Observação: no momento, o Seek com streaming não é recomendado, pois você não pode usar a pontuação de confiança, nem obter quaisquer campos de carga útil como url. Estaremos trabalhando nisso com a equipe da Watson. Novos modelos de embedding e a capacidade de usar um modelo de embedding personalizado com detecção de intenção NS e mAIstro
    • Melhorias na Tradução! A tradução NS agora é até 80% mais rápida para grandes traduções.
    • LLMs hospedados no NeuralSeek. Ao usar um plano BYO-LLM, agora fornecemos um LLM base hospedado globalmente (mistral-7b) e um LLM de tradução especialmente construído para uso com esse plano, sem cobranças adicionais, apenas a cobrança normal de Seek se aplica. Isso deve facilitar muito o início com o NS.

    Junho - 2024

    Novos recursos

    • Novas plataformas suportadas:
      • vLLM / motores de inferência genéricos estilo openAI. Isso permite que você conecte e use muitos mais motores de inferência on-prem e SaaS
      • O Google Vertex agora é suportado, e adicionamos Gemini 1.5 pro e Flash. Esses modelos são bastante bons - o Pro está no mesmo nível que o GPT-4o, Claude3 Sonnet e Mistral-Large
    • Atualizações do mAIstro!
      • Gráficos embutidos. Com um LLM compatível, você pode solicitar que um gráfico seja gerado como parte da saída.
      • Saída formatada - gerar e exibir HTML e javascript
      • Nova visualização Raw - veja o código por trás da geração de gráficos e HTML
      • Saída em PDF
      • Menus de Hover! Quando no construtor visual, todos os nós agora permitirão que você veja e insira quaisquer segredos, variáveis ​​definidas pelo usuário e pelo sistema ou gere uma nova variável. Facilita muito a construção!

    • Integração nativa ao watsonx.governance. No mAistro, veja nosso modelo de exemplo de como configurar isso - é realmente fácil, apenas 3 etapas. Para watsonx.governance, você só precisa de uma chave de API do IAM e, do seu espaço de implantação de Produção em x.gov, em ações/informações do modelo, precisamos do seu ID do data mart de Avaliação e do ID de Assinatura. Enviaremos todas as medidas do NeuralSeek para watsonx.governance para que você possa coletá-las e governá-las entre instâncias e mostrar uma história de governança mais ampla. Também fornecemos uma integração aberta caso você queira fazer algo mais personalizado.

    • Novas integrações do mAIstro: (existem tantas funções nativas e conectores no mAistro agora que tivemos que adicionar um recurso de pesquisa!)

      • Jira
      • Trello
      • Github
      • Slack
      • AWS S3
      • Pesquisas na web do Google/Bing/Yahoo/DuckDuckGo.

    • Ferramentas JSON: Adicionamos filtro de matriz JSON e Escape JSON para facilitar o trabalho com cargas úteis complicadas dentro do mAIstro.

    • Auto-Escape. Agora, ao usar o editor visual do mAistro, escaparemos automaticamente quaisquer aspas. Isso deve facilitar a construção no mAistro para usuários de negócios. Descobrimos que essas atualizações, juntamente com o auto-construtor do mAistro que lançamos no mês passado, reduzem muitos casos de uso para funcionar fora da caixa, sem modificações adicionais necessárias aos fluxos gerados automaticamente.

    • Atualizações de governança: Aprimoramos a guia Insights de Token e adicionamos um novo gráfico Resolução de Perguntas à guia Visão Geral para ajudar a rastrear quantas perguntas estão atingindo seu limite mínimo de confiança.

    • A guia Logs agora sinaliza respostas que tiveram ações de PII, ativação de HAP e injeção de prompt.

    Maio - 2024

    Novos recursos

    • Bases de conhecimento virtuais! Agora você pode usar o mAistro para definir um fluxo e usá-lo como uma base de conhecimento virtual. Quer consultar várias instâncias de descoberta de uma só vez? Fácil. Elastic e DB2 e mesclar os resultados? Fácil. Raspar algumas páginas da web ao vivo e usar essas? Fácil. Veja o novo modelo no mAistro para um exemplo de como configurar isso.

    • Lista de permissões semântica (Configuração / Ajuste do modelo semântico). Especifique palavras ou frases a serem excluídas de penalidades semânticas.

    • Atualizações de curadoria. Agora, as respostas geradas pelo uso de um filtro exibirão o filtro usado durante a geração.

    • Traduções personalizadas. Faça o upload de um arquivo de treinamento via API.

    Recursos do mAistro

    • Processamento de imagens / suporte multimodal no mAistro. Agora você pode pegar imagens da web, arquivo local ou Google Docs e fluxo-las por meio de LLMs que suportam processamento de imagens (Claude3, GPT-4, GPT-4o). Veja o novo modelo de exemplo. E sim, você pode alimentar o Seek com base em imagens se usar isso com a KB virtual!

    • Auto-construtor para mAistro (SaaS - apenas). Você tem sido sobrecarregado ou com medo de tentar o mAistro? Não está claro sobre como construir algo? Agora o modal de boas-vindas (e o modal de Carregar) irá perguntar a você apenas para descrever seu caso de uso e, em seguida, geraremos automaticamente um modelo personalizado para você.

    • Conector Snowflake! Agora disponível no mAistro.

    Recursos de governança

    • Insights de Token! Um novo módulo chega à Governança do NeuralSeek (planos BYO-LLM apenas). Obtenha insights de custo sobre o uso de seu LLM, métricas sobre a velocidade de geração, comparações de custo com LLMs de capacidade semelhante. É muito convincente.

    • Atualizações de governança - agora você pode rastrear o percentual de acertos de cache e respostas editadas na guia Insights Semânticos.

    Novos modelos

    • Muitos novos. GPT-4o, Mixtral8x-22 e mais.

    Abril - 2024

    O lançamento da Governança do NeuralSeek.

    Novos recursos

    • Remover alucinações - ative isso via a guia Configurar em Pontuação Semântica. Como parte de uma resposta Seek, remova qualquer frase contendo uma palavra-chave (nome próprio, entidade) que não esteja contida em sua documentação de origem.

    • Propostas. Nossa visão sobre versionamento / mudanças de configuração. Agora você pode definir uma configuração como uma Proposta e, em seguida, chamar essa proposta dinamicamente da API ou das guias Seek ou Home. Isso ajuda a separar a configuração administrativa das alterações propostas pelos especialistas. Também permite que você execute várias configurações ao mesmo tempo sem passar por uma substituição completa toda vez. Atualize uma configuração e clique em Propor Alterações Além disso, um novo recurso Registrar Configurações Alternativas - permite que você bloqueie a curadoria de respostas provenientes dessas propostas, para que você possa testar de forma isolada em uma única instância. Título e Descrição da Configuração - como parte do nosso módulo de Governança e do lançamento de propostas, agora solicitaremos um título e uma descrição da configuração ao salvar. Esses fluxos entram no lado da governança da casa para explicabilidade.

    • Suporte Pinecone - nosso lançamento inicial. mais opções de modelo de incorporação estão chegando em breve.
    • Conector KB Milvus. Então você agora pode fazer pesquisa vetorial no watsonx.data
    • Retornar Documentos Completos - estamos implementando a capacidade de você retornar um documento completo em vez de um trecho. Atualmente lançado para Discovery e AppSearch. Dessa forma, se você criou ou pré-recortou cuidadosamente sua documentação, você pode garantir que o documento completo seja retornado.
    • Melhorias de desempenho - algumas grandes atualizações em áreas como raspagem dinâmica da web, divisão da janela de contexto e muito mais.

    Recursos do mAIstro

    • Segredos! - defina variáveis na guia Configurar para ocultá-las dos usuários normais do mAIstro. Os usuários on-premises também podem definir variáveis no nível do sistema operacional. Muito útil para passar / ocultar informações de conexão do banco de dados.
    • Loop de Contexto - divida um grande bloco de texto por tokens e faça um loop sobre ele. Muito útil para traduzir grandes documentos ou enviar coisas grandes por um pequeno LLM. Veja o exemplo de Tradução de Documentos no mAIstro
    • Conector do Google Drive - puxe e grave em um Google Drive
    • Loop de Variáveis - faça um loop em uma matriz de dados

    Recursos de Governança

    • Módulo de Governança. Nosso foco inicial com este primeiro lançamento é uma visão holística da governança RAG com filtragem baseada em tempo e Intenção/Categoria. Estaremos implementando muito mais recursos adicionais nas próximas semanas aqui. No lançamento, temos:
      • Gráficos de visão geral executiva
      • Análise de Intenções - quais intenções estão em tendência e como elas estão se saindo - regressão de modelo / documento
      • Desempenho do Sistema - monitore sua instância e compare com o universo NS
      • Insights semânticos - Qual é a qualidade das respostas geradas
      • Insights de Documentação - Qual documentação é mais usada e como ela está se saindo
      • Insights de Configuração - monitorar alterações de configuração e acompanhar a rotatividade ao longo do tempo

    Novos Modelos

    • LLama 3 - um grande avanço em relação ao llama 2 em termos de sua capacidade de seguir instruções. No watsonx, a janela de contexto é pequena, no entanto, o mixtral ainda é melhor no geral.
    • jais-13b-chat - no watsonx frankfurt, para casos de uso em árabe
    • granite-7b-lab - Este parece melhor que os outros modelos granite. Por baixo dos panos, é baseado no llama-2...
    • Mistral-Large - similar e iterativamente melhor que o mixtral. ainda não disponível no watsonx.

    Março - 2024

    O Explore agora é renomeado para mAIstro e ganhou uma variedade de novos recursos.

    Novos recursos

    • RAG totalmente personalizado agora disponível no NeuralSeek, oferecendo simplicidade via Seek e complexidade via mAIstro, tudo pronto e sem código necessário.

    Recursos do mAIstro

    • Curar: Envie suas próprias perguntas e respostas para as guias curadoria, análise e log.
    • Categorizar: Conecte-se ao categorizador NS para obter categoria e intenção.
    • Cache de Consultas: Verifique e retorne respostas curadas e editadas.
    • Pontuação semântica: Acesse o modelo de pontuação semântica a partir de um fluxo Maistro.
    • Extrair gramática: Extraia entidades, substantivos, datas e muito mais do texto.
    • Adicionar contexto: Lembre-se da última parte da conversa e injete o assunto anterior no texto (para uma chamada de KB ou LLM).
    • Parar: Pare a execução (útil para condicionais).
    • Truncar por tokens: Recorte o texto por um número definido de tokens LLM (use isso para reduzir sua documentação de KB para caber na janela de contexto LLM).

    Novos modelos

    • Dois novos modelos adicionados ao watsonx no NeuralSeek: Granite 7B Japanese e Elyza Japanese Llama.

    Outras atualizações

    • Nova introdução walk-me adicionada para ajudar novos usuários a começar no mAIstro.

     

    Fevereiro - 2024

    Novos recursos

    • Filtragem/mascaramento de PII pré-LLM: Remova ou mascare informações de identificação pessoal (PII) antes de enviar consultas para um Banco de Conhecimento (KB) ou LLM. Use elementos pré-construídos ou adicione os seus próprios usando expressões regulares.
    • Detecção de injeção de prompt: A entrada do usuário é pontuada em relação a um modelo interno para identificar possíveis tentativas de injeção de prompt. Palavras problemáticas são filtradas e toda a entrada pode ser bloqueada com base na probabilidade de injeção de prompt.
    • Tradução de KB multilíngue: Ao especificar um idioma de saída diferente do idioma do KB, a entrada do usuário agora pode ser traduzida automaticamente para o idioma do KB para obter melhores respostas.
    • Esquemas arbitrários para Explore: O NeuralSeek Explore agora suporta esquemas arbitrários, permitindo que os usuários o conectem a qualquer coisa que envie uma solicitação POST, processem-na e a retornem no formato correto. Esse recurso permite a reformulação dinâmica de mensagens com base no contexto salvo, no histórico de conversas ou em outros critérios, proporcionando uma experiência mais personalizada para os usuários.
    • Atualizações na Mitigação de Injeção de Prompt: O recurso de teste agora exibe as pontuações de diferentes frases elegíveis para serem removidas da entrada do usuário, aprimorando os recursos de detecção de injeção de prompt.

    Novos modelos

    • A watsonx.ai introduz o modelo Granite-20b-5lang-instruct-rc em versão prévia e vários novos modelos são adicionados ao Bedrock.

    Melhorias no Explore

    • Salvaguardas como Filtro de Palavrões e Injeção de Prompt agora estão disponíveis no Explore.
    • Vários novos modelos de exemplo foram adicionados para demonstrar esses novos recursos.
    • Os usuários agora podem modificar o modelo de Personalização WA fornecido nos exemplos na guia Explore para reformular dinamicamente as mensagens que fluem pelo Watson Assistant, oferecendo uma experiência de chatbot mais personalizada.
    • Os parâmetros de cabeçalho overrideschema e templatename na API do Explore permitem uma configuração e personalização fáceis de esquemas no Explore, possibilitando uma integração perfeita com vários sistemas e aplicativos.

     

    Janeiro - 2024

    Novos recursos

    • Trabalhos de execução em threads paralelas foram introduzidos no Explore, permitindo uma execução mais rápida de modelos complicados, muitas vezes superando a codificação personalizada em Python.
    • Melhorias na busca de várias etapas: os usuários agora podem controlar o número de etapas anteriores enviadas para o LLM para uma experiência mais semelhante ao ChatGPT.
    • Melhorias na extração:
      • Suporte para definição de tipos de entidade regex e palavras-chave, reduzindo a carga de trabalho em LLMs menores/menos capazes e melhorando a velocidade de extração.

    Melhorias no Explore

    • Conectores diretos para vários bancos de dados, incluindo Postgres, Oracle, MySQL, MariaDB, MS SQL e Redshift.
    • Variáveis de sistema para injetar data, hora, UUIDs, números aleatórios, etc.
    • Funcionalidade 'Extrair' adicionada ao Explore.
    • Gerador de modelo OpenAPI do Explore aprimorado para uma integração mais fácil com o Watsonx Assistant.
    • Novos modelos disponíveis, incluindo Custom RAG, Causa de Perda de Seguro e Lógica Condicional.
    • Opção de especificar o LLM a ser usado nas etapas de Explore LLM para evitar atingir os limites de taxa e distribuir a carga de maneira eficaz.

    Atualizações

    • Permissões de usuário mais refinadas: os usuários agora podem conceder acesso a guias, restringindo a capacidade de gravação de guias específicas.
    • Todos os idiomas estão agora desbloqueados, permitindo que os usuários utilizem o NeuralSeek com qualquer idioma suportado pelo LLM escolhido.
    • Funcionalidade de Parar/Cancelar para Seek e Explore durante as respostas de streaming.

     

    Dezembro - 2023
